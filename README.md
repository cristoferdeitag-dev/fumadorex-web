# 🚭 Fumadorex Web

Sitio oficial de **Fumadorex** — Método psicológico para dejar de fumar.

## Stack
- [Astro](https://astro.build/) — Framework web ultrarrápido
- Tailwind CSS (próximamente)
- Hosting: Hostinger

## Desarrollo local
```bash
npm install
npm run dev
```

## Deploy
Push a `main` → Hostinger actualiza automáticamente.
